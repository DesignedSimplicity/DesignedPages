MetaGraffiti Spacetime Continuum Tracking Repository

A .NET SDK to process geographic and topographic data for map visualizations
The development genesis of MetaGraffiti originated from a travel blog I started when I set off on a trip to circumnavigate the planet in 2010. I wanted to keep track of the places I visited, and more importantly, geotag my photos so they could be displayed on a map and show a progress of my adventures.

DesignedNet Microsoft .NET Code Generation Framework
GitHub

Automated generation of n-tier web application directly from a database schema
The DesignedNet Framework provides an enterprise level architecture and codebase to streamline development efforts of web based applications using the Microsoft .NET Framework. The architecture consists of a robust code generation engine that instantly produces multiple application layers consistent with the Microsoft Solution Framework methodology and n-tier application design concepts.
